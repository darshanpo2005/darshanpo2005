### 📊 ZeroTrace — Privacy-First Cookieless Analytics

<div align="center">

[![ZeroTrace](https://img.shields.io/badge/📊%20ZeroTrace-Cookieless%20Analytics-059669?style=for-the-badge)](https://github.com/darshanpo2005/zerotrace)

</div>

> **Production-grade distributed analytics platform — GDPR compliant, no cookies, no PII**
> Paste a JS snippet → track visitors anonymously → real-time dashboard

| Feature | Details |
|---------|---------|
| 🔒 Privacy | HMAC-SHA256 anonymous session hashing, zero PII stored |
| ⚡ Speed | Sub-10ms event ingestion via async Celery queue |
| 🗄️ Database | ClickHouse (columnar) for sub-100ms analytics queries |
| 🤖 ML Layer | 5 attribution models: First touch, Last touch, Linear, Time decay, Position |
| 📈 Dashboard | Real-time KPIs, funnel analysis, traffic sources, device breakdown |
| 🐳 Stack | FastAPI + ClickHouse + PostgreSQL + Redis + Celery + React + Docker |

[![View ZeroTrace](https://img.shields.io/badge/View%20Project-ZeroTrace-059669?style=for-the-badge&logo=github)](https://github.com/darshanpo2005/zerotrace)
